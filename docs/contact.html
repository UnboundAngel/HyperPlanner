Report a Security Issue

Found a security vulnerability? Please report it responsibly:

Security Reports

Email security issues directly to security@hyperplanner.today. Please do not disclose security vulnerabilities publicly before we've had a chance to address them. We take all security reports seriously and will respond within 4 hours.

We appreciate responsible disclosure and may offer recognition or bounties for significant findings.
