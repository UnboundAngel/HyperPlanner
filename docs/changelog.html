v1.0.0
April 1, 2025

Initial Release

HyperPlanner launches with the following core features:

  • New Smart Capture - Natural language task entry with date, context, priority, and tag parsing.
  • New Flexible Panel System - Drag, resize, and arrange panels to create your perfect workspace.
  • New Calendar View - Day, week, and month views with drag-and-drop scheduling.
  • New Kanban Board - Visual task management with customizable lanes.
  • New Focus Duo - Split view for today's tasks and backlog.
  • New Unified Task/Event Model - Seamlessly drag items between views.
  • New Recurrence Rules - Daily, weekly, monthly, and custom repeat patterns.
  • New Keyboard-First Design - Complete keyboard navigation and shortcuts.
  • New Offline-First Architecture - Works without internet, stores data locally.
  • New Glassy Aesthetic - Beautiful frosted glass design with warm accents.
  • New Progressive Web App - Install on any device from your browser.

Thank You!

Thank you to all our beta testers who helped shape HyperPlanner. Your feedback made this launch possible.
