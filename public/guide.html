Setup & Onboarding

  • Local-first start: Boots offline. Data lives in localStorage until you sign in to sync.
  • Demo vs full: Demo blurs AI, Automations, Templates, Sync, Insights. Everything else works locally.
  • Panels & layout: Left capture, center calendar/board, right insights. Presets save per device.
  • Data model: Title, description, context, priority, due/start/end, recurrence, tags, status, location, subtasks.

Layouts & Panels

  • Presets: Minimal, Dense, Creative, Dashboard, Kanban-first, Focus Duo, Split, Airy.
  • Handles: Hover top-right dots to drag/hide; drop targets glow with glass outlines.
  • Calendar density: 60/30/15m; “now” line pulses in dark mode.
  • Board lanes: Inbox, In Progress, Done, Archived; rename/recolor via statuses.
  • Hero cards: Next up, Focus load, Alerts adapt to dominant context colors.

Theming & Visual System

  • Tokens: --hp-accent, --hp-accent-secondary, --hp-surface, --hp-glass, --hp-border, --hp-text, --hp-text-muted.
  • Glass rules: 7–12% glass, 18px blur, edge lighting, warm accent glows.
  • Modes: Light/dark morph, contrast guardrails for inputs and cards on mixed backgrounds.
  • Motion: 180ms hover, 220ms lift, 30–45s background drift, optional parallax hero.

Icons & Illustration

  • Packs: Minimal, Color chips, Mono. Semantic map: work, personal, study, travel, finance, health, focus, errands.
  • Overrides: Context overrides saved in localStorage; filenames match UI expectations.
  • Locked indicators: Padlocks + blur on icon labs and AI panels in demo mode.

Automations & NLP

  • NLP capture: Detects time phrases, explicit times, weekdays, “every” rules, contexts, priorities.
  • Custom NLP rules: Trigger → context/priority/due offset, applied before built-ins.
  • Automation rules: “When X → do Y” visual builder; demo shows locks/teasers.
  • AI: Optional, blurred in demo; unlock with account or API key.

Calendar, Tasks, Boards

  • Calendar: ISO weeks, configurable start/end; conflicts outline in soft red.
  • Tasks: Context icon + tags; descriptions ellipsize; location pin supported.
  • Boards: Drag ghost with blur shadow; drop highlights lanes; counts in headers.
  • Capture tips: “tomorrow 3pm”, “every monday”, “after lunch”, “in 45m”, “next week tuesday”, “daily 7am”.

Demo vs Full

  • Demo includes: Capture, calendar, boards, drag/drop, offline data.
  • Demo locks: Theme Lab, Automations, Smart Templates, Multi-device sync, Cloud backup, AI Suggestions, Insights dashboards.
  • Watermarks: Demo pill + upgrade banners encourage sign-up.

Performance & Accessibility

  • Offline-first: Local cache; sync queue flushes after sign-in.
  • Rendering: Lean shadows, few gradients per card, throttled parallax for low-power devices.
  • A11y: Focus rings in accent; respects prefers-reduced-motion; high-contrast text guardrails.
  • Keyboard reachability: All primary buttons are tabbable; skip-links planned.
  • Color safety: Automated contrast check for inputs on mixed glass/light surfaces.

Shortcuts & Micro-interactions

  • N: New task. T: Today. /: Search (planned). Cmd/Ctrl + K: Command palette (planned).
  • Drags: Cards scale to 0.98 on drag; drop zones thicken borders.
  • Hover: Accent glow on primary CTAs; ghost buttons remain flat.
  • Edge cases: Empty states show status lines; locked states show blur + padlock copy.
  • Tooltips: Locked tooltips read “Unlock by creating a free account.”

Security & Privacy

  • Local demo: No network calls in demo; mock data stored locally.
  • Sync: Requires sign-in; encrypted in transit. Per-device sessions.
  • Telemetry: Off by default; future opt-in only.
  • Exports: Planned JSON/ICS export from Settings.

Data schema references

  • Task fields: id, title, description, context, priority, due_date, start_time, end_time, recurring_rule, tags, status, location, subtasks.
  • Calendar slotting: start_of_day, density (60/30/15), conflict detection per slotKey.
  • Automations: trigger (string), action (context|priority|due), offset (minutes), value (string), id.
  • NLP rules: trigger → context/priority/due; stored in localStorage.

Copy & Upgrade UX

  • Locked CTA: “Unlock by creating a free account.”
  • Demo watermark: “Demo Mode — Changes Are Not Saved.”
  • Banners: Soft, non-blocking nudges near locked panels.
  • Download buttons: Disabled until release; show “Coming soon.”
  • AI disclaimers: Clear labels on AI-only panels; blurred previews in demo.

Roadmap (Public)

  • Command palette (Cmd/Ctrl + K) with actions for layouts, filters, navigation.
  • Template marketplace for shared routines and icon packs.
  • Webhooks for automations (condition/action builder).
  • ICS/CalDAV import/export.
  • Shared workspaces with per-panel visibility.
