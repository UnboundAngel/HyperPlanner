API

Planner API overview.

Local Express + SQLite with simple JSON endpoints.

← Back to landing

Core endpoints

  • GET/POST/PUT/PATCH/DELETE /api/tasks — CRUD tasks.
  • GET /api/smart-terms — seed terms for NLP.
  • GET/PUT /api/email-settings — email config.
  • POST /api/email-settings/test — send test email.

Notes

  • Auth is not enforced in demo; deploy with your own auth proxy.
  • SQLite file lives under server/planner.db; WAL enabled.
  • Rate-limit and add CORS rules before production use.
