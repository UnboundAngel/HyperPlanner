Power moves

  • Save multiple layouts per device for different work modes.
  • Use tags + contexts to drive future color-coding.
  • Inline snooze buttons in the task modal to keep today clear.
  • Keyboard shortcuts: N (new), T (today), Esc (close).

Under the hood

  • LocalStorage holds layout/theme/template prefs.
  • Calendar density is just CSS vars—safe to customize.
  • Email scheduler runs in the backend cron every minute/hour.
