Do this first

  • Try presets: Minimal, Dashboard, Kanban, Split.
  • Drag hero/quick capture/calendar/cards; hide anything via the handle menu.
  • Save your layout with “Save current” to remember per device.
  • Reset layout if things feel off.

Pro tips

  • Use smaller density for time-blocking (15m), larger for high-level planning.
  • Keep calendar centered and put boards or settings on the right for speed.
