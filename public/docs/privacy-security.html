Principles

  • Local storage by default; no cloud until you opt in.
  • Email alerts use your SMTP creds; nothing shared beyond that.
  • No analytics/telemetry by default.
  • Export/import planned so you can leave anytime.

Habits

  • Review email settings; disable if you don’t want off-device data.
  • Use strong SMTP passwords; rotate occasionally.
  • Run behind HTTPS and add auth if deployed.
