Sample phrases

  • “handoff tomorrow 3pm @work !high #project”
  • “in 45m call alex” → due in 45 minutes
  • “every monday 9am standup” → recurring weekly
  • “after lunch review deck” → default afternoon slot

Custom rules

  • Trigger → Context (e.g., “clinic” → health)
  • Trigger → Priority (e.g., “urgent” → high)
  • Trigger → Due offset (e.g., “eod+2h” → +120m)
