Patterns to try

  • "handoff tomorrow 3pm @work !high #launch" → fills due, context, priority, tag.
  • "call mom sat 10am @personal" → schedules weekend call.
  • "desk setup every monday 9am" → recurrence planned.
  • Macro phrases like “before lunch” or “next sprint” are recognized from the seed list.

Tips

  • Use @context, !priority, #tags inline; we strip and store them.
  • We seed smart terms from server/data/smart_terms_seed.json; extend there.
  • Highlights preview parsed fields before submit.
